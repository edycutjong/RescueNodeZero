**Project Description (Short):**
RescueNode Zero is a military-grade, completely offline triage intelligence hub. It uses Actian VectorAI DB to perform sub-15ms Reciprocal Rank Fusion searches across field reports, HAZMAT protocols, and drone imagery entirely on edge devices.

**Key Technologies:**
- Actian VectorAI DB
- Vector Embeddings (all-MiniLM-L6-v2)
- Next.js 16 (App Router) & React 19
- FastAPI & Python 3.12
- Tailwind CSS v4
