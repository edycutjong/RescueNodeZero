# 🎙️ RescueNode Zero — Video Narration Script

> ~2 minute walkthrough. Read naturally, not rushed.

## SCENE 1: Hook (0:00–0:08)

**[Screen: Black → fade in text "When the grid goes dark…" → dashboard fades in]**

> "When a disaster strikes, cloud infrastructure is the first thing to die. Cell towers collapse. Data centers flood. And first responders — the people who need critical information the most — are left with absolutely nothing. No Google. No databases. No AI assistants. Nothing."

---

## SCENE 2: Command Center (0:08–0:25)

**[Screen: Full dashboard — offline badge, search bar, stats all visible]**

> "This is RescueNode Zero — an entirely air-gapped AI triage hub that runs on a single laptop. No cloud. No Wi-Fi. No external dependencies whatsoever."

> "At the top, you can see the system status: 49 pre-seeded documents, all indexed in Actian VectorAI DB. That green badge? It says 'AIR-GAPPED' — every single model runs locally."

---

## SCENE 3: HAZMAT Search — RRF Fusion (0:25–0:45)

**[Screen: Type "chemical burn treatment acetone" → results appear instantly]**

> "Let's run a search. A firefighter encounters a chemical spill — acetone. They need treatment protocols *now*."

> "Watch the latency badge. 0.9 milliseconds. That's Reciprocal Rank Fusion — combining semantic vector search with keyword matching — all running locally on VectorAI DB. No round trip to any server. No internet required."

> "The results pull from both HAZMAT protocols and medical triage guides simultaneously. Decontamination procedures, PPE levels, UN hazard codes — all cross-referenced in under a millisecond."

---

## SCENE 4: Medical Triage (0:45–0:55)

**[Screen: Search "crush syndrome field extraction" → medical results]**

> "Now a field extraction scenario — crush syndrome. This is a time-critical medical emergency where every second matters."

> "RescueNode pulls the exact protocol: START triage classification, fluid resuscitation steps, tourniquet guidance — all instantly available even in a total communications blackout."

---

## SCENE 5: Multimodal Ingestion (0:55–1:15)

**[Screen: Upload Zone — drag image → drag audio file → new results appear]**

> "But text search is just the beginning. RescueNode Zero is fully multimodal."

> "I'm dropping a drone reconnaissance photo here. MobileCLIP — running locally — generates a caption describing the damage. That caption is vectorized and searchable alongside every other document."

> "Now a field audio report. Whisper transcriber — also local — converts the radio communication into text, extracts zone and reporter metadata, and indexes it. All searchable."

---

## SCENE 6: Safety Filters + Inventory (1:15–1:35)

**[Screen: Toggle allergen chips → results update → switch to inventory panel]**

> "Critical safety feature: allergen-aware filtering. If a patient is allergic to penicillin — toggle it on — and every protocol containing penicillin is automatically excluded from results. This prevents dangerous cross-reactions in the field."

> "And on the right, real-time inventory tracking. Watch those amber and red badges — 'LOW STOCK' on IV bags, 'CRITICAL' on epinephrine. Responders know exactly what supplies are available before they commit to a treatment plan."

## Production Notes

- **Tone**: Urgent but measured. Think field briefing, not sales pitch.
- **Pace**: Let the search results load naturally — the sub-ms latency is the demo itself.
- **Audio**: Record in quiet environment. If adding music, use dark ambient/cinematic tension — low BPM, no vocals.
- **Screen Resolution**: 1280×800 or 1920×1080 (matches dashboard viewport)
- **Total runtime**: ~1:45–2:00
