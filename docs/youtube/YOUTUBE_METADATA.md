## Video Description

⚡ RescueNode Zero — Air-Gapped Triage Intelligence Hub

When disasters strike, cloud infrastructure dies first. Cell towers collapse, data centers flood, and first responders lose access to every critical database. A firefighter facing an unknown chemical spill can't Google "chlorine gas PPE requirements" when there's no internet.

RescueNode Zero runs an entire AI-powered triage system on a single laptop — no cloud, no Wi-Fi, no dependencies. Sub-millisecond hybrid vector search across HAZMAT protocols, medical procedures, and field inventory. Powered by Actian VectorAI DB.

⏱️ Timestamps:
0:00 — Hook: When the grid goes dark
0:08 — Command Center Overview
0:25 — Live HAZMAT Search: "chemical burn treatment acetone"
0:40 — RRF Fusion: Semantic + Keyword in 0.9ms
0:55 — Medical Triage: Crush Syndrome Protocol
1:10 — Multimodal: Drone Image + Audio Ingestion
1:25 — Allergen Safety Filters (penicillin exclusion)
1:40 — Inventory Tracking with Critical Alerts
1:50 — Why Air-Gapped AI Matters

🔗 Links:
💻 GitHub: https://github.com/edycutjong/RescueNodeZero
🏆 Hackathon: https://dorahacks.io/hackathon/actian-vectorai

🛠️ Built with:
• Next.js 16 (App Router) + React 19
• Tailwind CSS v4 (Military SOC aesthetic)
• Python 3.12 + FastAPI (async backend)
• Actian VectorAI DB (local Docker vector search)
• all-MiniLM-L6-v2 (384-dim text embeddings)
• MobileCLIP ViT-B/32 (drone image captioning)
• openai-whisper base (field audio transcription)
• Reciprocal Rank Fusion (RRF) — hybrid search

📊 Key Metrics:
• 0.9ms average query latency
• 49 pre-seeded documents (HAZMAT, medical, inventory)
• 100% air-gapped — zero cloud API calls
• Multimodal: text + images + audio
• Allergen-aware safety filtering
